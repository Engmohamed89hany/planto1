# 🌿 Planto - Plant Shop Landing Page

Planto is a modern and responsive landing page for a plant shop built using **React + Vite**.  
The project focuses on clean UI design, responsive layout, and smooth user experience.

---

## 🚀 Live Demo
👉 https://planto1.netlify.app/

---

## 🧰 Tech Stack

- React.js
- Vite
- CSS / Tailwind 
- JavaScript (ES6+)

---

## 🎯 Features

- Modern and clean UI design
- Fully responsive layout (mobile, tablet, desktop)
- Fast performance using Vite
- Smooth section-based landing page
- Reusable components structure
- Ready for deployment on Netlify
